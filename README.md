# Site para Previsão do Tempo

## 📝 Apresentação:
Projeto para verificação do tempo onde o usuário pode olhar o clima em qualquer cidade pelo mundo em tempo real.
API consumida: HG Weather (https://hgbrasil.com/status/weather/)
